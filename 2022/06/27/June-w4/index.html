June 4th Week - Getting ready for Early Access release

27 Jun 2022

I have decided on a Steam release date for the game. Haven’t announced it yet because I’m not sure if I can make it D: but I think I can, unless some horrible problem comes up. This week I wrote some story for chapter 2 (I should have done this sooner). I wanted to release the Early Access version with the entirety of chapter 2, but I am not sure I can make it. Which means that early access might launch with half of chapter 2. I have two reasons for that:

1) I did some major rebalancing of the game’s Soul Crush system last week, to test that stuff I need some new simple dev tools 2) The game had some issues for people with big screens and I added that last week 3) The steam page still has screens and trailer from the last version. I might do something about this, not sure.

I decide to rebalance the game now because I want people to be able to import their saves. The rebalance means that advance people will lose their Soul Crush / Challenge Area progress. Figured it was better do that sooner than later.

You can try the game here: https://pidroh.github.io/GenericRPGIdle/
